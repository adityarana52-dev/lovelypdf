# Play Store Listing Template

## App Name

DocPDF

## Short Description

Scan documents, make clear A4 PDFs, and share instantly.

## Full Description

DocPDF helps you scan documents with automatic edge detection and convert them into clean A4 PDF files.

Features:

- Scan 1 to 30 pages
- Auto edge detection and crop support
- Color, grayscale, and black-and-white PDF modes
- Clear PDF output with balanced file size
- Share instantly to WhatsApp, email, and other apps
- Temporary PDF storage with automatic cleanup

DocPDF is designed for quick document scanning and easy sharing without keeping unnecessary files on the device for too long.
